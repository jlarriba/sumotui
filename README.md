# sumotui
Sumo TUI is a text console application to display Sumo tournament (basho) player stats

# Quickstart
```
$ pip3 install -r requirements.txt
$ python3 sumo_tui.py 202601-13
```
202601 -> basho
13 -> day

You'll see:
1. Loading screen with "Loading 1/21: Wrestler1 vs Wrestler2" progress
2. Once loaded, instant navigation with ↑/↓ keys
3. Press r to refresh data, q to quit
